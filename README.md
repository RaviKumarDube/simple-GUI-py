# Simple GUI Python

A lightweight and intuitive Python wrapper designed to make Graphical User Interface (GUI) development fast, clean, and accessible.

## 🚀 Overview

**simple-GUI-py** simplifies the process of building desktop applications. Instead of managing complex classes and nested callbacks, you can define your interface using simple Python structures and an easy-to-follow event loop.

## ✨ Features

*   **Minimalist Code**: Build functional windows in under 10 lines of code.
*   **Intuitive Layout**: Define your UI using familiar Python lists.
*   **Cross-Platform**: Consistent behavior across Windows, macOS, and Linux.
*   **Ready-to-Use**: Perfect for automation tools, data entry forms, and small utilities.
